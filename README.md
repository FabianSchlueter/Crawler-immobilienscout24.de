# Crawler-immobilienscout24.de

This crawler goes through all results of a previously defined search for real estate on the website www.immobilienscout24.de.

By this you can get data (about 100 features) for all real estate offers at once.

## How to use:
Specify a search on www.immobilienscout24.de manually. Paste the url of the first result page in the code.

Specify the path where to store the crawled data.

Run the file.
